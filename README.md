# 🌿 MyCare

O **MyCare** é um aplicativo voltado para o acompanhamento e organização da rotina de cuidados com a pele. A plataforma está sendo desenvolvida para auxiliar usuários na criação de hábitos saudáveis de skincare, oferecendo uma experiência personalizada de acordo com as características e objetivos de cada pessoa.

Com foco em praticidade e bem-estar, o aplicativo permite gerenciar produtos, acompanhar o uso diário e receber lembretes importantes, contribuindo para uma rotina mais consistente e eficiente.

## ✨ Principais Funcionalidades

- Personalização de acordo com o tipo de pele;
- Definição de objetivos de cuidados;
- Gerenciamento de produtos de skincare;
- Controle de validade dos produtos;
- Acompanhamento da rotina diária;
- Histórico de utilização;
- Notificações e lembretes personalizados.
